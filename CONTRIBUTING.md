# Contributing to Password Health Tracker

## Code of Conduct

We are committed to providing a welcoming and inspiring community for all. Please read and adhere to our Code of Conduct.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/yourusername/password-health-tracker.git`
3. Create a feature branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Commit: `git commit -am 'Add your commit message'`
6. Push: `git push origin feature/your-feature-name`
7. Create a Pull Request

## Development Guidelines

### Code Style

**Python (Backend):**
- Follow PEP 8
- Use type hints
- Maximum line length: 100 characters
- Use Black for formatting

**JavaScript (Frontend):**
- Use ES6+ syntax
- Follow Airbnb style guide
- Use Prettier for formatting
- Maximum line length: 100 characters

### Commit Messages

Follow the Conventional Commits format:
```
type(scope): description

body (optional)

footer (optional)
```

Types: feat, fix, docs, style, refactor, perf, test, chore

### Testing

- Write unit tests for all new features
- Maintain >80% code coverage
- Run tests before submitting PR: `pytest` or `npm test`

### Documentation

- Update README.md if adding features
- Add docstrings to Python functions
- Add JSDoc comments to JS functions
- Update API_DOCUMENTATION.md for API changes

## Pull Request Process

1. Update documentation and README
2. Add tests for new functionality
3. Ensure all tests pass
4. Update CHANGELOG.md
5. Request review from maintainers
6. Wait for approval and status checks to pass

## Reporting Bugs

Use GitHub Issues with the following template:
- Description
- Steps to reproduce
- Expected behavior
- Actual behavior
- Screenshots (if applicable)
- Environment details

## Feature Requests

Use GitHub Issues with the following template:
- Description
- Use case
- Proposed solution
- Alternative solutions

## Questions?

Feel free to reach out:
- GitHub Discussions
- Email: support@passwordhealthtracker.com
- GitHub Issues

Thank you for contributing!
